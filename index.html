Fleet Architecture

🧠 PLATO Reasoning

Structured 5-atom decomposition: premise → reasoning → hypothesis → verification → conclusion. Every agent decision traced and logged.

📡 Fleet Communication

Inter-agent messaging via keeper. Agents query each other's state, send bottles (messages), and coordinate without human intervention.

🦀 Hermit Crab Model

Agents are crabs, repos are shells. Agents hop between shells as capabilities evolve. Shells (repos) persist and carry context forward.

⚓ Dojo Training

Greenhorn agents come in under-skilled, produce real value while learning. Leave each project more capable than they arrived.
